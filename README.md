# Matrix Project
This project is a Computer application that performs basic matrix operations
> The project is implemented by **_C++_** programming language
## Operations
| **Operation** | **Description** |
| - | - |
| Show Matrix Value | `A` |
| Additions | ` C = A + B` |
| Subtraction | ` C = A - B` |
| Multiplication | ` C = A * B` |
| Transpose | ` C = A'` |
| Division | ` C = A / B` |
